## Usage
- Open PowerToys Run and type `ms`, then your search.
- To search for specifics use:
- ! = Title
- \# = Text
- % = Folder
- | = Tag

## Installation


## Configuration
- Open PowerToys Settings > PowerToys Run > Plugins > `PwrMassCode`.
	- You can change the plugin keyword
	- Masscode url default is: `http://localhost:4321/`.
	- Change the prefixes used for "advanced" search


### Visual Studio
- Open the `PwrMassCode.csproj` (or add it to a solution)
- Select `x64` or `ARM64` platform
- Build Debug/Release
- Run debug.ps1 as admin to deploy, or manually copy files
	- gsudo .\debug.ps1
